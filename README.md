# npuzzle
42 project for solving a sliding block puzzle using the A* algorithm.

To run (on MacOS)
```virtualenv npuzzle```

```source npuzzle/bin/activate```

```pip install -r requirements.txt```

```python3 gen.py -u 3 | python3 solver.py && open -a Google\ Chrome index.html```
